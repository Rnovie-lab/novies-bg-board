Scheduling
PDF Import
Auto-detection
Upload a shooting schedule PDF and the parser automatically extracts every shooting day, scene, and BG count — no manual entry required. Supports all major scheduling program PDF formats.
Organization
Scene Strips
Drag to reorder
Each day shows its scenes as strips — like a physical production board. Grab the ⠿ handle and drag to reorder scenes within a day. Add, edit, or remove scenes and roles at any time.
Compliance
SAG Tracking
Minimum compliance
Set a SAG minimum per day (e.g. 25 for TV Basic Agreement). Live counters show SAG vs. Non-Union and flag a warning if the minimum isn't met. Green confirmation when you're clear.
Revision Management
Import New Draft
Smart revision merge
When a revised schedule arrives mid-prep, import it over your existing work. BG Board compares the new draft against your current breakdown and shows exactly what changed — your existing BG roles, bumps, and notes are fully preserved on all unchanged scenes.
New Scenes Added to the schedule — select to bring in
Dropped Scenes No longer on the board — choose to remove
Moved to New Day Scene shifted to a different shoot day
Location Changed Set or description updated in new draft
Each change is individually selectable — apply only what you need.
Compensation
Bump Library
SAG schedule rates
A built-in library of SAG-AFTRA bump rates — wardrobe maintenance, formal wear, vehicles, props, cosmetic, hazardous work, wet/snow. Select from the list or add a custom bump. All amounts roll into the per-person pay calculation automatically.
Projects
Save & Export
Named projects
Save named snapshots at any time — multiple saves per episode. Duplicate a project to start a new episode with the same stand-in roster. Export generates a print-ready PDF breakdown sorted by day with grand totals.
Stand-Ins
Roster — Setup Tab
Add recurring stand-ins in Setup → Stand-Ins Roster. Enter name, character covered, rate, and default hours. They appear on every day automatically.
Per-Day Toggle
Each day shows the full roster. Uncheck "Working" to mark a stand-in off for that day. Adjust hours per day independently — minimum 8 hours enforced.
+ Day SI Button
Need a one-day fill-in? Hit + Day SI on any day in Breakdown — no Setup required. Day-only SIs count toward the SI total and show a DAY badge.
Roles & Rates
Every BG role carries a tier, base rate, hours, and bumps. Pay auto-calculates including 1.5× OT after 8 hrs and 2× after 12. Reuse tags flag roles appearing in multiple scenes.
⚡ Quick BG Entry
Hit ⚡ Quick BG Entry on any day to type roles in plain text: (10) Park extras 12hrs — all roles created instantly without opening each scene individually.
◈ Reuse Tags
If the same BG type appears in multiple scenes, a Reuse tag appears automatically — useful for flagging wardrobe continuity or recurring character types across a shooting day.
⎙ Print Breakdown
Go to Export tab → Print / Save as PDF. Choose Save as PDF in your print dialog. Lists every day's SI roster, scenes, roles, bumps, and running totals.
